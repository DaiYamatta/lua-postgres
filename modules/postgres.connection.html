Module "postgres.connection"

A Postgres connection.

Functions

Functions

execute (sql, params)
Executes a query.

Parameters:

  • sql: An SQL query
  • params: A single parameter, or a table of parameters

Example:

local result, error = conn:execute("SELECT * FROM users WHERE name = $1", "joe")
local result, error = conn:execute("SELECT * FROM users WHERE name = $1 AND age < $2", {"joe", 100})

Return values:

  1. result - A result userdata or nil on failure.
  2. message - If execution failed, an error message
open ()
Opens the connection to the Postgres server.

Example:

local ok, err = conn:open()
if err then error(err) end

Return values:

  1. status - Whether or not the connection succeeded.
  2. message - If the connection failed to open, an error message.
