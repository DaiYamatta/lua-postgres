connection (connection_string)
Creates a new connection object. Note that this does not immediately connect to the database, it simply creates a handle. You can then invoke connection.open to make the connection immediately. If you do not open the connection manually, lua-postgres will establish a connection the first time it is needed.

Parameters:

  • connection_string: A Postgres connection string. See the documentation for libpq for the full list of values that can be in the string.

Example:

local conn = postgres.connection('dbname=my_db host=localhost user=postgres')

Return value:

A Postgres connection
